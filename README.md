# AI Todo List 

An AI-powered Todo List application built using:
- HTML, CSS, JavaScript
- Node.js + Express
- Ollama (Local LLM)

## Features
- Add / delete todos
- AI-generated task lists
- Multi-task planning
- Modern UI
- No database, no login

## Run Locally
```bash
npm install
node server.js
